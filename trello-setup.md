# Настройка Trello для системы Кайдзен

## Пошаговая инструкция

### Шаг 1: Создайте доску

1. Откройте [trello.com](https://trello.com)
2. Нажмите **"Создать доску"** в меню слева
3. Название: `Кайдзен [год]` (например: `Кайдзен 2024`)
4. Выберите фон (рекомендую тёмный)

### Шаг 2: Настройте цветные метки

Перейдите в меню доски → **Метки** и создайте 5 цветов:

| Цвет | Метка | Значение |
|------|-------|----------|
| Жёлтый | Личностное развитие | Книги, курсы, здоровье |
| Фиолетовый | Семья | Семейные дела, отношения |
| Синий | Бизнес | Работа, проекты, финансы |
| Розовый | Общество | Волонтёрство, помощь другим |
| Оранжевый | Хобби | Увлечения, отдых, творчество |

### Шаг 3: Создайте годовую доску

Создайте доску с названием `Год [год]` с такими списками:

```
Нужно сделать | Запланировано | Январь | Февраль | ... | Декабрь | Сделано
```

**Как работает:**
- **Нужно сделать** — все задачи, которые прошли фильтры
- **Запланировано** — задачи с привязкой к дате
- **Месяцы** — карточки раскладываются по месяцам
- **Сделано** — выполненные задачи

### Шаг 4: Создайте месячную доску

Для каждого месяца создайте доску `Месяц [месяц] [год]`:

```
Нужно сделать | Запланировано | В процессе (мяч у меня) | В процессе (у другого) | Сегодня | Сделано
```

**Правила карточек:**

| Список | Когда карточка здесь |
|--------|---------------------|
| Нужно сделать | Задача ещё не начата |
| Запланировано | Есть конкретная дата |
| В процессе (мяч у меня) | Я работаю над задачей |
| В процессе (у другого) | Жду ответа от кого-то |
| Сегодня | Делаю именно сегодня |
| Сделано | Задача выполнена |

### Шаг 5: Создайте доску привычек

Доска: `Привычки [год]`

Списки: по неделям или по месяцам

**Как использовать:**
1. Создайте карточку для каждой привычки
2. Каждый день отмечайте галочкой или нет
3. Используйте **чек-листы** для отслеживания

### Шаг 6: Создайте доску встреч

Доска: `Встречи`

Списки:
```
Входящие | На этой неделе | Запланировано | Процесс | Сделано
```

### Шаг 7: Доска для сложных проектов

Доска: `Проекты`

Для каждого большого проекта (стройка, переезд и т.д.) — отдельный список.

## Правила работы

### Одна мысль = одна карточка

Не объединяйте задачи. Каждая мысль — отдельная карточка.

### Делегирование

Когда поручаете задачу:
1. Напишите в карточке: **"Исполнитель: [имя]"**
2. Переместите в **"В процессе (у другого)"**
3. Установите **напоминание** через 3 дня

### Мяч на стороне исполнителя

Если вы ждёте ответа — карточка в "В процессе (у другого)". Вы НЕ виноваты, что задача стоит.

### Материализация мыслей

Если мысль пришла — сразу создавайте карточку. Не держите в голове.

## Полезные функции Trello

### Дедлайны
Добавляйте даты к карточкам. Trello покажет, когда задача горит.

### Чек-листы
Для больших задач — разбивайте на подзадачи с галочками.

### Описание
Пишите контекст задачи в описании карточки.

### Комментарии
Оставляйте заметки о прогрессе.

### Вложения
Прикрепляйте файлы, скриншоты, ссылки.

## Ежедневный ритуал

### Утро (15 минут)
1. Откройте доску **"Сегодня"**
2. Перекиньте карточки из **"Запланировано"** в **"Сегодня"**
3. Проверьте **привычки**

### Вечер (5 минут)
1. Выполненные карточки → **"Сделано"**
2. Невыполненные → решите: перенести или оставить на завтра
3. Новые мысли → сразу создавайте карточки

## Дополнительные приложения

Для полной системы Кайдзен используйте 3 приложения:

| Приложение | Для чего | Платформа |
|------------|----------|-----------|
| **Trello** | Управление задачами | Веб + мобайл |
| **The Ultimate Calendar** | Цветное планирование дня | iOS |
| **Offline Habits** (или аналог) | Трекер привычек | iOS + Android |

### The Ultimate Calendar
- Платное приложение (~$5)
- Цветовая кодировка по категориям
- Вид: день / неделя / месяц / год
- Синхронизация с календарём

### Трекер привычек
Любое приложение для трекинга:
- **Habitica** (игровой формат)
- **Loop Habit Tracker** (бесплатное, Android)
- **Streaks** (iOS)
- **Productive** (iOS + Android)

## Частые ошибки

### Не делайте:
- Не объединяйте карточки (одна мысль = одна карточка)
- Не ставьте слишком много задач на день (максимум 3-5)
- Не держите задачи в голове
- Не пропускайте фильтры

### Делайте:
- Каждое утро — планирование (15 мин)
- Каждый вечер — ревизия (5 мин)
- Используйте цветные метки
- Делегируйте, что можно
- Отмечайте сделанное

## Шпаргалка

```
Хочу / Должен → Миссия → 5 направлений → Канбан → Сегодня
```

1. **Выгрузка** — всё из головы в карточки
2. **Хочу/Должен** — разделить своё и чужое
3. **Миссия** — делает счастливым? полезно обществу? среде?
4. **5 направлений** — распределить по сферам
5. **Канбан** — раскидать по месяцам
6. **Сегодня** — выбирать 3-5 задач на день
