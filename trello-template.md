## 1. Годовая доска

### Списки:
```
Нужно сделать → Запланировано → Январь → Февраль → ... → Декабрь → Сделано
```

### Шаблон карточки:
```
[Цветная метка сферы]
[Название задачи]
---
Дедлайн: [дата]
Приоритет: высокий / средний / низкий
```

---

## 2. Месячная доска (пример: Январь 2024)

### Списки:
```
Нужно сделать → Запланировано → В процессе (мяч у меня) → В процессе (у другого) → Сегодня → Сделано
```

### Шаблон карточки:
```
[Цветная метка сферы]
[Название задачи]
---
Контекст: [описание]
Исполнитель: [имя или "я"]
Дедлайн: [дата]
Статус: 🟡 начато / 🔵 в работе / 🟢 готово
```

### Примеры карточек:

**Бизнес (синяя):**
```
Подготовить презäsentацию для клиента
---
Контекст: Клиент ждёт до 15 января
Исполнитель: я
Дедлайн: 14 января
Статус: 🟡 начато
```

**Семья (фиолетовая):**
```
Записаться к стоматологу
---
Контекст: Регулярный осмотр
Исполнитель: я
Дедлайн: 20 января
Статус: ⬜ не начато
```

**Личностное развитие (жёлтая):**
```
Прочитать "Атомные привычки"
---
Контекст: Книга на 300 страниц
Исполнитель: я
Дедлайн: конец января
Чек-лист:
- [ ] Глава 1-3
- [ ] Глава 4-6
- [ ] Глава 7-9
- [ ] Глава 10-12
```

## 4. Доска встреч

### Списки:
```
Входящие → На этой неделе → Запланировано → Процесс → Сделано
```

### Шаблон карточки встречи:
```
🤝 Встреча с [имя]
---
Дата: [дата и время]
Место: [адрес / Zoom]
Цель: [зачем встречаемся]
Подготовить: [что взять/сделать]
Итог: [записать после встречи]
```

### Пример:

```
🤝 Встреча с Джаныбеком
---
Дата: 15 января, 14:00
Место: кафе "Руми"
Цель: Обсудить партнёрство
Подготовить: Презентацию, визитки
Итог: (заполнить после)
```

---

## 5. Доска проектов

### Списки:
```
Идеи → В работе → На паузе → Готово
```

### Шаблон карточки проекта:
```
🏗️ [Название проекта]
---
Описание: [что делаем]
Срок: [дата окончания]
Бюджет: [если есть]
Команда: [кто участвует]
---
Чек-лист:
- [ ] Этап 1: [описание]
- [ ] Этап 2: [описание]
- [ ] Этап 3: [описание]
- [ ] Финал: [описание]
```

## Цветовые метки (создайте в настройках доски)

| Цвет | Метка | Использование |
|------|-------|---------------|
| Жёлтый | Личностное развитие | Обучение, книги, здоровье |
| Фиолетовый | Семья | Семейные дела, дети, родители |
| Синий | Бизнес | Работа, проекты, финансы |
| Розовый | Общество | Волонтёрство, помощь другим |
| Оранжевый | Хобби | Увлечения, отдых, спорт |
| Зелёный | Срочно | Горящие задачи |
| Красный | Важно | Ключевые задачи |

## Правила именования карточек

### Формат:
```
[Эмодзи] [Глагол] [Объект] [Контекст]
```

### Примеры:
- 📚 Прочитать "Атомные привычки" (к январю)
- 💼 Подготовить отчёт для руководства (до пятницы)
- 🏃 пробежать 5 км (в субботу утром)
- 🤝 Встретиться с партнёром (в кафе "Руми")
- 🏗️ Начать ремонт ванной (март-апрель)

---

## Система статусов

### Эмодзи для статусов:
- ⬜ Не начато
- 🟡 Начато (первые шаги)
- 🔵 В работе (активно делаю)
- 🟠 На паузе (ждём кого-то)
- 🟢 Готово
- ❌ Отменено

---

## Еженедельный ревью (каждое воскресенье)

1. Откройте доску текущего месяца
2. Пересмотрите все карточки в "Нужно сделать"
3. Определите 3-5 главных задач на неделю
4. Перекиньте их в "Запланировано"
5. Проверьте доску привычек — все ли отмечены?
6. Посмотрите "Сделано" — отметьте прогресс

---

## Ежемесячный ревью (последний день месяца)

1. Все выполненные карточки → в "Сделано" годовой доски
2. Невыполненные → решить: перенести или отменить
3. Посмотрите баланс цветов — нет ли перекоса?
4. Определите 3 главных цели на следующий месяц
5. Обновите доску следующего месяца

---

## Экспорт и резервная копия

Trello позволяет экспортировать данные:
1. Меню доски → **Ещё** → **Экспортировать в JSON**
2. Сохраняйте раз в месяц
3. Храните в облаке или на диске

## Горячие клавиши Trello

| Клавиша | Действие |
|---------|----------|
| `N` | Новая карточка |
| `E` | Редактировать карточку |
| `C` | Добавить метку |
| `D` | Добавить дедлайн |
| `F` | Архивировать |
| `Q` | Мои карточки |
| `/` | Поиск |
