# Кайдзен Планировщик

Веб-приложение для личной продуктивности на основе философии Кайдзен (непрерывное совершенствование).

**Ссылка на видео-источник:** https://youtu.be/OUUTNGuPjQM

---

## Что реализовано

### Основной функционал

| Функция | Описание | Статус |
|---------|----------|--------|
| Выгрузка головы | Запись всех мыслей на карточки | Готово |
| Фильтры | Хочу/Должен → Миссия → 5 направлений | Готово |
| Канбан-доска | 6 колонок: Нужно → Запланировано → В процессе → Сегодня → Сделано | Готово |
| Трекер привычек | Ежедневное отмечание, стрики, график 6 недель | Готово |
| Вкладка "Сегодня" | Задачи на сегодня + прогресс-бар | Готово |
| Миссия (Икигай) | Настройка жизненной цели | Готово |

### Продвинутый функционал

| Функция | Описание | Статус |
|---------|----------|--------|
| Pomodoro-таймер | 25/5 мин, сохраняется при перезагрузке | Готово |
| Режим фокуса | Полноэкранный режим для одной задачи | Готово |
| Таймтрекинг | Замер времени на каждой карточке | Готово |
| Повторяющиеся задачи | Ежедневно/еженедельно/ежемесячно | Готово |
| Undo (отмена) | Ctrl+Z — отмена последних 25 действий | Готово |
| Поиск | Глобальный поиск по всем карточкам | Готово |
| Toast-уведомления | Красивые всплывающие сообщения | Готово |

### Экспорт/импорт

| Формат | Описание | Статус |
|--------|----------|--------|
| JSON | Полный бэкап всех данных | Готово |
| Markdown | Документ со всеми задачами | Готово |
| PDF | Печатная версия через браузер | Готово |

### Дизайн

| Элемент | Описание | Статус |
|---------|----------|--------|
| Notion-style | Светлая тема, чистый минимализм | Готово |
| Sidebar | Вертикальная навигация (220px / 56px) | Готово |
| Inter шрифт | Современная типографика | Готово |
| Micro-animations | Плавные переходы и hover-эффекты | Готово |
| Glassmorphism | Модалки с backdrop blur | Готово |
| Адаптивность | Desktop + Tablet + Mobile | Готово |

### Горячие клавиши

| Клавиша | Действие |
|---------|----------|
| `1-6` | Переключить вкладку |
| `Ctrl+Z` | Отменить |
| `Ctrl+K` или `/` | Поиск |
| `Ctrl+E` | Экспорт JSON |
| `Ctrl+I` | Импорт JSON |
| `Ctrl+P` | Экспорт PDF |
| `N` | Новая карточка |
| `DblClick` | Фокус-режим |
| `← →` | Фокус: навигация |
| `Space` | Фокус: таймер |
| `Esc` | Закрыть модал/фокус |

## Что еще можно реализовать

### Высокий приоритет

| Функция | Сложность | Описание |
|---------|-----------|----------|
| **Command Palette (Cmd+K)** | Средняя | Универсальное окно поиска/действий как в Linear/Notion |
| **Тёмная тема** | Простая | Переключатель светлая/тёмная тема |
| **Drag & drop в фильтрах** | Средняя | Улучшить перетаскивание карточек между колонками фильтров |
| **Повторяющиеся привычки** | Средняя | Автоматическое создание задач по расписанию для привычек |

### Средний приоритет

| Функция | Сложность | Описание |
|---------|-----------|----------|
| **Шаблоны проектов** | Средняя | Готовые шаблоны: марафон, курс, стройка и т.д. |
| **Метки/теги** | Простая | Произвольные теги помимо 5 направлений |
| **Приоритеты задач** | Простая | Высокий/средний/низкий с цветовой кодировкой |
| **Дедлайны с напоминаниями** | Средняя | Уведомления о приближении дедлайна |
| **Привязка к дате** | Средняя | Конкретные даты для задач в Канбан |
| **Сортировка карточек** | Простая | По дате, приоритету, категории |

### Низкий приоритет

| Функция | Сложность | Описание |
|---------|-----------|----------|
| **Экспорт в календарь** | Средняя | .ics формат для Google/Apple Calendar |
| **Интеграция с Trello API** | Сложная | Синхронизация с реальными досками Trello |
| **Мультипользователь** | Сложная | Совместная работа над задачами |
| **Статистика за месяц/год** | Средняя | Расширенная аналитика с графиками |
| **Резервное копирование** | Простая | Автоматический экспорт по расписанию |
| **Темы оформления** | Средняя | Несколько цветовых схем на выбор |
| **PWA** | Средняя | Установка как приложение на телефон |
| **Оффлайн-режим** | Простая | Уже работает через localStorage |

### Идеи из видео

| Идея | Реализация |
|------|-----------|
| Продавец/Покупатель | Каждый процесс имеет "клиента" — следующий этап |
| Воронка/труба | Пропускная способность = задачи/день |
| Материализация мыслей | Осуществлено через "Выгрузку головы" |
| Графики динамики | 6-недельный график привычек |

## Технологии

- **HTML5** — структура
- **CSS3** — стили (переменные, grid, flexbox, animations)
- **JavaScript** — логика (ES6+, localStorage)
- **Без зависимостей** — всё в одном файле

---

## Установка

1. Скачайте `kaizen-planner.html`
2. Откройте в браузере (Chrome/Edge/Firefox/Safari)
3. Начните использовать!

Данные сохраняются в localStorage браузера.
